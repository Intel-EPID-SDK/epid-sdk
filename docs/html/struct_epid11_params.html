Detailed Description

Intel(R) EPID 1.1 Parameters.

Intel(R) EPID 1.1 parameters are: (p, q, h, a, b, coeff, qnr, orderG2, p', q', h', a', b', g1, g2, g3). The size of the Intel(R) EPID public parameters of G1, G2, G3, and GT is 6464 bits or 808 bytes.
